# Mix Parlay Tracker

Track your mix parlay bets - picks, results, bankroll & win rate.

## Features
- Add parlay picks (1X2, AH, O/U, Odd/Even)
- Update match scores - auto evaluate win/lose
- Track bankroll & profit/loss
- Stats per market, per leg
- History with full detail

## Build
Uses GitHub Actions to build APK in the cloud (no local SDK needed).

Push to `main` branch → APK auto-built → download from Actions artifacts.

## Tech
- Flutter 3.24
- Provider (state management)
- Material 3 (dark theme)
